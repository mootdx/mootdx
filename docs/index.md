## 项目概述

Mootdx 是一款纯Python语言开发的类似TradeX的行情数据接口的实现。

## 项目特点

- 基于 pytdx 二次封装。
- 完全支持`3.6+`
- 以及全平台 `Windows / MacOS / Linux`
- 更加友好的API接口
- 自动匹配最优服务器

## 运行环境
 - 操作系统: `Windows / MacOS / Linux` 都可以运行.
 - Python: 3.5 以及以上版本, 不支持 python2.
 - 依赖库: pytdx>=1.67

## 快速安装

```bash
$ pip install -U mootdx
```
