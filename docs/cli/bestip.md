## 01. 查看命令帮助

```shell
mootdx bestip --help

Usage: mootdx bestip [OPTIONS]

  测试行情服务器.

Options:
  -l, --limit TEXT   显示最快前几个，默认 5.
  -t, --tofile TEXT  将数据输出到文件.
  -w, --write        将最优服务器IP写入配置文件 ~/.mootdx/config.json.
  -v, --verbose
  --help             Show this message and exit.

```
